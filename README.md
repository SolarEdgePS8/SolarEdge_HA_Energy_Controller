# SolarEdge HA Energy Controller

Portabler Home-Assistant-Controller für SolarEdge-Batteriespeicher mit vier eigenständig abgesicherten Betriebsarten:

- Eigenverbrauch maximieren
- Netzdienlich laden
- Akku schonen
- EVOpt optimiert

> **Projektaufbau läuft.** Der erste geprüfte Release Candidate wird derzeit auf der Referenzinstallation getestet. Code und vollständige Dokumentation werden zunächst über einen Draft Pull Request vorbereitet.

Das Projekt ist unabhängig von privaten Fahrzeug-, Wallbox-, Wärmepumpen-, Strompreis- oder Reserve-Automationen.
