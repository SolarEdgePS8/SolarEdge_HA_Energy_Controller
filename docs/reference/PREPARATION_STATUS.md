# Vorbereitungsstand

Der Branch `agent/prepare-v0.1.0-rc.2` dient der Vorbereitung des ersten öffentlichen Release Candidates.

Bereits vorbereitet:

- verständliche Einstiegshilfe;
- Voraussetzungen;
- Erstinstallation;
- Entity-Mapping;
- erster Start;
- Update und Migration;
- Fehlerdiagnose;
- Datenschutz und Schreibsicherheit;
- getrennte Dokumentation der vier Betriebsarten;
- Integrationsanleitungen für SolarEdge, PV-Prognose, Wetter, SQL, evcc und EVOpt;
- spätere Ergänzung des geprüften RC2-Codes nach Abschluss des Aktivierungstests.

Der Branch ist noch nicht zur produktiven Installation freigegeben.
